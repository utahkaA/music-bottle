Music Bottle
============
Music Bottle Firmware at OKLAB hard seminar.

How to use
----------
(適当。暇なときにちゃんと書く。)
- なんか飲み物の入ったカップなどを Music Bottle の上に置くと音楽が鳴る。
- カップの飲み物の量によって音楽が変わる。
- 君も Music Bottle とスタバに行って Let's ドヤァ！
